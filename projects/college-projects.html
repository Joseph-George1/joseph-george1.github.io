College Projects

A collection of various academic projects developed throughout computer and communication engineering studies.

Repository Overview

This repository is a collection of various academic projects developed collaboratively by students. Each project demonstrates skills, teamwork, and commitment to learning in fields such as computer engineering, embedded systems, and operating systems.

Repository Structure

Line_follower_car/

Arduino-based line following robot project using IR sensors for path detection and motor control.

  • Line_follower_car.ino - Arduino source code
  • Arduino-Line-Following-Robot.png - Project schematic
  • Arduino_Line_Follower_Project.pdf - Project documentation
  • cs.pptx - Presentation slides

Microprocessors_project/

8086 Assembly language Stopwatch implementation with real-time display capabilities.

  • mycode.asm - Assembly source code
  • 8086 Stopwatch.pdf - Project report
  • 8086 Stopwatch.docx - Documentation
  • 8086_Stopwatch_Report.pptx - Presentation

OS_Project/

First-Come, First-Served (FCFS) CPU Scheduling algorithm implementation in Python with visualization.

  • fcfs.py - Python implementation
  • input.txt - Sample input data
  • FCFS_Scheduling_Report&explain.pdf - Detailed report
  • FCFS.pptx - Presentation slides

Thoutha - Teeth Management System

A comprehensive platform connecting dental students and patients in Egypt, facilitating dental education and healthcare access.

Project Overview

Thoutha is a comprehensive dental management platform designed to bridge the gap between dental students seeking clinical experience and patients looking for affordable dental care in Egypt. The system facilitates dental education by helping students find diverse cases for their graduation projects while providing patients with quality dental services.

The platform features both web and mobile applications, making it accessible to users across different devices and preferences.

Key Features

  • Web application built with React for seamless browser-based access
  • Mobile application developed with Flutter for iOS and Android
  • Intelligent chatbot for patient assistance and appointment scheduling
  • Comprehensive booking system for dental appointments
  • Secure user authentication and profile management
  • Case matching algorithm to connect students with appropriate patients
  • Real-time notifications and appointment reminders
  • Patient history and records management

Technology Stack

This project leverages modern technologies to deliver a robust and scalable solution:

Dart/Flutter (57%) JavaScript (12.7%) HTML (11.5%) Python (5%) CSS (4.9%) Shell (3.7%) Spring Boot AI Chatbot
